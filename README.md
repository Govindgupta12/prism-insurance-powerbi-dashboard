# 📊 Prism Insurance Analytics Dashboard (Power BI)

An interactive Power BI dashboard designed to analyze and visualize key business metrics for **Prism Insurance Pvt. Ltd.** This project demonstrates data visualization, business intelligence, and storytelling using Power BI.

---

##  Project Objective

The objective of this dashboard is to provide actionable insights into the insurance business by analyzing:

* Policy performance
* Customer distribution
* Revenue trends
* Claims data

This helps stakeholders make informed and data-driven decisions.

---

## 📌 Key Features

* 📍 Interactive slicers for dynamic filtering
* 📈 KPI tracking (Revenue, Policies, Claims)
* 🌍 Region-wise and category-wise analysis
* 👥 Customer segmentation insights
* 📊 Clean and intuitive dashboard design

---

## 🛠️ Tools & Technologies Used

* Microsoft Power BI Desktop
* Data Modeling
* DAX (Data Analysis Expressions)
* Data Visualization Techniques
